# Resultat Alert

Ce dossier regroupe les livrables de cadrage pour les alertes et les rapports GEO Arctic.

Fichiers fournis :

- `alert-types.md` : catalogue des alertes utilisateur.
- `alert-types.xlsx` : version tableur du catalogue d alertes.
- `report-types.md` : catalogue des rapports generes et recommandes.
- `report-types.pdf` : version PDF de synthese des types de rapports.

Perimetre :

- alertes produit et alertes techniques,
- rapports operationnels et rapports de pilotage,
- formats de sortie utiles pour produit, client, SEO/GEO et direction.
